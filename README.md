# zach_capstone

Exploring the Intersection of Alternative Energy and CO2 Emissions
This project aims to explore the relationship between alternative energy adoption and CO2 emissions reduction using data-driven analysis and methodologies.

Installation
Clone the repository to your local machine.

Usage
Data Sources
The project utilizes datasets from various reputable sources, including [World Bank Open Data](https://data.worldbank.org/) and the [IEA](https://www.iea.org/), to examine trends in alternative energy adoption and CO2 emissions across different countries.

Dependencies
Python 3.7+
Pandas
Seaborn
Matplotlib
Statsmodels

Getting Started
Run the Jupyter notebook in the notebooks directory to analyze and visualize the data.
Follow the step-by-step instructions provided in the notebooks to explore different aspects of the data.

Features
Analyze global CO2 emissions trends over time.
Explore alternative energy adoption patterns and their impact on CO2 emissions.
Investigate regional differences in alternative energy adoption and emissions reduction.
